# 배달 서비스 플랫폼

고객, 점주가 함께 사용하는 배달 서비스 백엔드 프로젝트입니다.
Spring Boot 기반으로 회원 관리, 주문, 배달 기능을 제공합니다.

---

## 기술 스택

- **Backend*: Spring Boot 3, Spring Security6, JPA, QueryDSL, JWT
- **DataBase*: MariaDB, Amazon RDS
- **Infra & Deploy*: Docker, Github Actions, AWS EC2
